Veil Stack (Canteen)

Docker container scheduler on FEVM + libp2p

Veil Stack schedules Docker containers through a smart contract on FEVM and a libp2p peer-to-peer mesh. The goal is to make every workload originate a Filecoin storage deal anchored on-chain — turning container orchestration into a programmatic deal origination engine.
